# CraftPlus
Adds recipes to your Minecraft world.
## Contribute
Feel free to help.

Example crafting recipe:
```
{
  "type": "minecraft:crafting_shaped",
  "pattern": [
    " W ",
    "WRW",
    " W "
  ],
  "key": {
    "W": {
      "tag": "minecraft:logs"
    },
    "F": {
      "item": "minecraft:furnace"
    }
  },
  "result": {
    "item": "tutorial:smoker",
    "count": 1
  }
}
```
